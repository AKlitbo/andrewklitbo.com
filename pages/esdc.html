During my time at ESDC I worked on the Homeless Individuals and Families Information System (HIFIS 4) software with a multi-disciplinary team comprised of quality assurance, programmers, technical writers, database administrators and non-technical team members. We applied and used Joint Application Development (JAD) techniques to receive input and implement changes based on user feedback.

Responsibilities:

  • Development of the HIFIS 4 software using C#, ASP.NET MVC 4.0, .NET 4, SQL Server 2008 R2, HTML, CSS, JavaScript, jQuery, XML, and JSON.
  • Development of design diagrams and database logic using SQL Server Management Studio 2012.
  • Development of business logic using Windows Communication Foundation (WCF) Services, Entity Framework 4 (EF), and LINQ.
  • Consuming web services for development of presentation layer logic using the Model-View-Controller (MVC) design pattern.
  • Development of presentation layer web pages using the Razor view engine, HTML, CSS, JavaScript, and jQuery.
  • Worked with various front-end frameworks including Bootstrap 3, Web Experience Toolkit (WET) 3.1, and WET 4.
  • Assisted in the upgrade from WET 3.1 to WET 4.
  • Used Team Foundation Server (TFS) as the primary source control and issue tracking system as well as GitHub for WET related changes.
  • Collaboration with the documentation and content teams to develop a custom content management system (CMS) for integrated software help.
