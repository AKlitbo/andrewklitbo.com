As a Programmer at Instrux Media, I am the developer of external business software solutions, such as the Project Status Application. This tool enables instant and auditable communication between the Instrux Media team and our clients. I am also responsible for the development of out internal portal to manage both company and client user accounts which doubles as the API Endpoint for the Status Application.

Other responsibilities include the lead development of accessible and compliant eLearning interactions using HTML, CSS and JavaScript. I am also engaged in the accessibility testing to comply with WCAG 2.0 AA and Section 508 for our interactive eLearning courses.

Responsibilities:

  • Development of internal web portal using CakePHP 3.2, MySQL, OpenLDAP, HTML, CSS, JavaScript and jQuery.
  • Maintenance and creation of company user accounts using PHP and OpenLDAP.
  • Development of database logic and database systems using MySQL and phpMyAdmin.
  • Migration of existing web based tools into internal web portal.
  • Development and maintenance of client reporting and communication tools using AngularJS 1.5.8.
  • Development of RESTful API using PHP for the client communication tool.
  • Development of accessible User Interface (UI) elements that meet Web Content Accessibility Guidelines (WCAG) 2.0.
  • Development of Adobe Dreamweaver CC templates for production team.
  • Development of accessible eLearning course content using HTML, CSS, JavaScript and jQuery from storyboards.
  • Troubleshooting issues with internal applications and eLearning course framework.
  • Used Subversion (SVN) as the primary source control tool.
  • Used npm (Node Package Manager), Grunt and Composer to maintain and build project dependencies.
  • Worked with various front-end frameworks including Bootstrap 3, Material Design Lite (MDL) and the Web Experience Toolkit (WET-BOEW) 4.
